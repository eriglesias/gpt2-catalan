# Contributing to gpt2-catalan

This is a small, focused research/learning repository quality over quantity is valued.

**All code must be written by hand.**

The entire point of this project is to understand it  the forward pass, the backward pass, the data pipeline ... 

## Code style

- Python: follow PEP 8. Use type hints where they add clarity.
- Keep functions small and single-purpose.
- SOLID, KISS  and other good code principles.
